## Overview

This uses OpenAI's GPT LLM to write git commit summaries

        
## Setup

      export OPENAI_API_KEY=xxxxxxxxxxxxxxxxxx

### Usage        

      git commit -m "$(ghostwriter)"

### Example

      $ echo "test" > test.txt"
      $ git add test.txt
      $ ghostwriter
      "Adds a test.txt file to repo"
